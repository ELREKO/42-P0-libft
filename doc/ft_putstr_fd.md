# ft_putstr_fd

## Description
Output the string 's' 

## Declaration 
```c
	void ft_putchar_fd(char *s, int fd)
```

## Parameter 
- s: The string to output. 
- fd: the file descriptor on which to write 

## Return 
- The trimmed string.
- NULL if the allocation fails.
